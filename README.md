# DataBase Engine

Este es un motor de base de datos desarrollado para el curso de Base de Datos 2, que implementa la creación de indices para busquedas eficientse en un sistema relacional.
Además contamos con busquedas en texto y búsqueda en multimedia(imagen). 

Vea el video del funcionamiento del programa en su primera fase(proyecto 1) [aquí](https://drive.google.com/file/d/19dgbM95GYSWHEaix2SkzLYBn6xnmU9xD/view?usp=sharing)

Vea el informe completo del programa(Wiki de GitHub) [aquí](https://github.com/DB2-UTEC/DataBaseEngine/wiki)

# Uso
El uso es sencillo, solo se tiene que clonar y ejecutar docker-compose up 
```bash
# Clonar el repositorio
git clone https://github.com/DB2-UTEC/DataBaseEngine.git
cd DataBaseEngine

# Levantar los contenedores
docker compose up
```
Y en su navegador de preferencia buscar: localhost:5173
